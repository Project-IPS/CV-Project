# Object Tracking with counter 


## Installation

Start with [**Python>=3.8**](https://www.python.org/) environment.


```
git clone https://github.com/Project-IPS/CV-Project.git
cd CV-Project
pip install -v -e .
```


## snippet to run
```
python examples/track.py --yolo-model yolov8.pt --source 0 --show
```
